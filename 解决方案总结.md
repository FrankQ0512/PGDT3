# 数据合并问题解决方案总结

## 问题诊断

### 原始错误
```
variables gvkey fyear do not uniquely identify observations in the master data
r(459);
```

### 根本原因
1. **Firm.csv**: 存在8个重复的gvkey-fyear组合
2. **CEO.csv**: 每个公司年份有多个高管记录，需要筛选CEO
3. **Board.csv**: 需要从董事级别数据聚合到公司级别
4. **Linking.csv**: 存在多对多映射关系

## 解决方案

### 1. 数据清理策略

#### Firm数据去重
- **方法**: 保留每个gvkey-fyear组合中数据最完整的记录
- **结果**: 117,028行 → 117,020行 (移除8行重复)
- **验证**: 0个重复组合剩余

#### CEO数据筛选
- **方法**: 筛选ceoann='CEO'的记录，重复时保留薪酬最高的
- **结果**: 114,660行 → 21,502行CEO记录
- **验证**: 0个重复CEO记录

#### 董事会数据聚合
- **方法**: 按公司年份计算独立董事比例
- **结果**: 99行董事记录 → 108个公司年度独立性指标

#### 链接表去重
- **方法**: 处理多对多关系，保留最佳匹配
- **结果**: 12,273行 → 11,090行

### 2. 变量构建

#### 核心变量
- **LEV** (资产负债率): 82,271个有效观测 (70.3%)
- **CASH** (现金持有率): 81,512个有效观测 (69.7%)
- **Overconfidence** (CEO过度自信): 21,383个有效观测 (18.3%)
- **B_INDEP** (董事会独立性): 24个有效观测 (0.02%)

#### 控制变量
- SIZE, ROA, TANG, MTB, COVID
- 交互项: OC_x_B_INDEP

### 3. 分析框架

#### 6个主要回归模型
1. **LEV模型1**: LEV ~ Overconfidence + 年份固定效应
2. **LEV模型2**: LEV ~ Overconfidence + B_INDEP + 年份固定效应
3. **LEV模型3**: LEV ~ Overconfidence + B_INDEP + OC_x_B_INDEP + 控制变量 + 年份固定效应
4. **CASH模型1-3**: 同样的递进结构

#### 稳健性检验
- 替代的CEO过度自信指标
- 子样本分析（按公司规模分组）
