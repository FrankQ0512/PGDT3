# CEO过度自信与公司融资政策分析 - Stata代码说明

## 项目概述

本项目使用Stata实现了CEO过度自信对公司融资政策影响的实证分析，包括董事会独立性的调节作用研究。代码基于四个主要数据源：Compustat财务数据、ExecuComp CEO薪酬数据、BoardEx董事会数据以及相应的链接文件。

## 数据文件说明

### 1. Firm.csv (Compustat财务数据)
- **观测数**: 117,028行
- **主要变量**: 
  - `gvkey`: 公司标识符
  - `fyear`: 财务年度
  - `at`: 总资产
  - `che`: 现金及等价物
  - `dlc`, `dltt`: 短期和长期负债
  - `ni`: 净利润
  - `csho`, `prcc_f`: 流通股数和股价

### 2. CEO.csv (ExecuComp CEO薪酬数据)
- **观测数**: 114,660行 (其中约21,536条CEO记录)
- **主要变量**:
  - `gvkey`: 公司标识符  
  - `year`: 年度
  - `ceoann`: CEO标识
  - `opt_unex_exer_est_val`: 未行权期权价值
  - `tdc1`: 总薪酬

### 3. Board.csv (BoardEx董事会数据)
- **观测数**: 99行
- **主要变量**:
  - `companyid`: 公司ID
  - `ned`: 是否为独立董事
  - `datestartrole`: 任职开始日期

### 4. Linking.csv (链接文件)
- **观测数**: 12,273行
- **作用**: 连接不同数据库的公司标识符

## 分析框架

### 核心研究问题
1. CEO过度自信如何影响公司杠杆率？
2. CEO过度自信如何影响公司现金持有？
3. 董事会独立性是否调节上述关系？

### 关键变量构建

#### 因变量
- **杠杆率** (`leverage`): 总负债/总资产
- **现金持有比率** (`cash_ratio`): 现金/总资产

#### 核心解释变量
- **CEO过度自信** (`overconfidence`): 基于期权持有价值比率的标准化指标
- **董事会独立性** (`board_independence`): 独立董事占董事会比例

#### 控制变量
- **公司规模** (`size`): ln(总资产)
- **盈利能力** (`roa`): 净利润/总资产
- **成长机会** (`mb`): 市账比
- **资产有形性** (`tangibility`): 固定资产/总资产

## 分析步骤详解

### 第1-4步: 数据清理和合并
1. **财务数据清理**: 删除关键变量缺失、总资产非正值的观测
2. **CEO数据处理**: 筛选CEO记录，处理同公司同年多CEO情况
3. **董事会数据**: 计算董事会独立性和规模指标
4. **数据合并**: 通过gvkey和年份合并，使用链接文件连接董事会数据

### 第5-6步: 样本筛选和数据预处理
- **样本筛选条件**:
  - 必须有CEO薪酬数据
  - 关键财务变量完整
  - 分析期间2010-2020年
  - 排除金融和公用事业公司
- **缩尾处理**: 对连续变量进行1%和99%分位数缩尾

**预期最终样本量**: 约10,500个观测值

### 第7-8步: 变量构建和描述性统计
- 构建年份和行业固定效应虚拟变量
- 生成CEO过度自信与董事会独立性交互项
- 计算主要变量的描述性统计

### 第9步: 主回归分析

#### 杠杆率模型
- **模型1**: 基础模型 (CEO过度自信 + 控制变量)
- **模型2**: 加入董事会独立性
- **模型3**: 加入交互项 (调节作用)

#### 现金持有模型  
- **模型4-6**: 对应杠杆率的三个模型规格

### 第10步: 稳健性检验

1. **替代测量**: 使用高期权持有虚拟变量
2. **分样本分析**: 按董事会独立性水平分组
3. **固定效应**: 公司固定效应回归
4. **内生性处理**: 使用滞后变量
5. **倾向得分匹配**: 控制选择偏差

### 第11-12步: 结果输出和可视化
- 生成回归结果表格
- 创建关系图表
- 计算边际效应
- 输出到Excel格式

## 预期研究发现

基于理论和现有文献，预期结果：

1. **CEO过度自信与杠杆率**: 正相关关系
   - 过度自信的CEO倾向于高估项目收益，更愿意使用债务融资

2. **CEO过度自信与现金持有**: 负相关关系  
   - 过度自信的CEO认为外部融资成本较低，倾向于持有较少现金

3. **董事会独立性的调节作用**:
   - 高独立性董事会能够约束CEO过度自信的负面影响
   - 交互项系数预期为负（杠杆率）或正（现金持有）

## 技术特色

### 计量方法
- **聚类标准误**: 在公司层面聚类以控制序列相关
- **固定效应**: 年份和行业固定效应控制不可观测异质性
- **稳健性检验**: 多种方法验证结果稳健性

### 数据处理
- **缺失值处理**: 合理的删除和填充策略
- **异常值处理**: 系统性缩尾处理
- **样本选择**: 明确的筛选标准和理由

## 使用说明

### 环境要求
- Stata 14.0或更高版本
- 足够内存处理大型数据集

### 运行步骤
1. 将四个CSV文件放在工作目录下
2. 调整代码中的路径设置
3. 运行 `CEO_Overconfidence_Analysis.do`
4. 查看生成的日志和结果文件

### 输出文件
- `CEO_Overconfidence_Analysis.log`: 完整分析日志
- `final_analysis_data.dta`: 最终分析数据集
- `leverage_results.rtf`: 杠杆率回归结果
- `cash_results.rtf`: 现金持有回归结果
- 关系图表文件

## 注意事项

2. **因果推断**: 虽然包含多种稳健性检验，但因果关系识别仍需谨慎解释
3. **时间跨度**: 分析期间可根据数据可得性调整
4. **行业分类**: SIC代码分类可能需要根据研究需要细化
